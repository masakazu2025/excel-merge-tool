# B-031 列・行フィルタ

## シナリオ

| ファイル | 内容 |
|----------|------|
| base.xlsx | Sheet1: 商品名・単価・数量・備考の4列 |
| file_b.xlsx | D列（備考）が全行「要確認」に一括変更 + B2・C3に個別変更あり |

## 確認手順

1. base.xlsx と file_b.xlsx をアップロードして比較
2. レポート画面を開く

## 確認ポイント

### D列フィルタ
- 「列 ▾」を開き、D列のチェックを外す
- → D列の差分が消える
- → 4行目（D列にしか差分がない）も消える

### 行フィルタ
- 「行 ▾」を開き、特定行のチェックを外す
- → その行の差分が消える

### Shift+クリック範囲選択
- チェックボックスまたはテキスト部分をクリック後
- Shift を押しながら別の項目をクリック
- → 範囲内の項目がまとめて切り替わる（テキスト反転なし）

## 変更内容

| セル | base | file_b |
|------|------|--------|
| B2 | 100 | 120 |
| C3 | 10 | 12 |
| D2 | 通常 | 要確認 |
| D3 | 通常 | 要確認 |
| D4 | 通常 | 要確認 |
