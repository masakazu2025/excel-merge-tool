# B-030 差分なしシートのタブを非表示にする

## シナリオ

| ファイル | 内容 |
|----------|------|
| base.xlsx | Sheet1（商品リスト）、Sheet2（担当者リスト）の2シート |
| file_b.xlsx | Sheet1は変更あり、Sheet2は変更なし（baseと同じ） |

## 確認手順

1. base.xlsx と file_b.xlsx をアップロードして比較
2. レポート画面を開く

## 期待する動作

- **Sheet1 タブのみ表示される**（Sheet2 は差分なしのため非表示）
- Sheet1 の差分: B2（単価 100→120）、C3（数量 10→12）の2件

## 変更内容

| セル | base | file_b |
|------|------|--------|
| Sheet1!B2 | 100 | 120 |
| Sheet1!C3 | 10 | 12 |
