# CHANGELOG

## v0.1.0 (2026-03-15)

### 追加

**バックエンド**
- Excel ファイル（.xlsx / .xlsm）の差分抽出（zipfile + ElementTree）
- 2ファイル・3ファイル比較に対応
- セル差分スキーマ：`changed_by`（b/c/both）+ `status`（new/delete/add/sub/update/conflict）
- 値は常に実値で保持（null で変更なしを表現しない）
- POST `/api/compare`：比較実行・結果を JSON で保存
- GET `/api/reports`：履歴一覧（ページネーション対応）
- GET `/api/reports/{id}`：レポート取得
- エラーハンドリング：E001〜E007（ファイル形式・破損・シート不一致・XML 破損・その他）

**フロントエンド**
- ホーム画面：DnD / ダイアログでファイル追加、比較元選択、比較実行
- レポート画面：差分グリッド（色分け）、詳細モーダル、キーボードナビゲーション、シートタブ、フィルター
- 履歴画面：一覧テーブル、ページネーション（件数切り替え）
- エラー表示：ネットワークエラー・APIエラー・JSON 不正

**テスト**
- バックエンド：pytest（73 件 PASSED、B-004/005/006 は次期実装のため Red 保留）
- フロントエンド：vitest + @testing-library（64 件 PASSED）
