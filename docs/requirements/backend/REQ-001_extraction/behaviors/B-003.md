# B-003 status を付与する

## 振舞

```
Given: 差分が検出されたセルがある
When:  差分抽出を実行する
Then:  各セルに status が付与される
```

## 判定ルール

| 条件 | status |
|------|--------|
| changed_by が "both" かつ b_value ≠ c_value | `"conflict"` |
| base_value が null → B or C に値あり | `"new"` |
| base_value に値あり → B or C が null | `"delete"` |
| b_value が base_value の末尾にテキストを追加した形 | `"add"` |
| b_value が base_value の末尾からテキストを削除した形 | `"sub"` |
| 上記以外の変更（途中編集・全置換・数式変更・日付変更など） | `"update"` |
| 変更なし | `"no_change"` |

## 備考

- テキスト diff 判定は Python の `difflib.SequenceMatcher` を使用
- `add` / `sub` の判定は末尾への純粋な追記・削除のみ対象（途中の変更は `update`）
- `conflict` は changed_by の判定より後に評価する
