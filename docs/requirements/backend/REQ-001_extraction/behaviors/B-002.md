# B-002 競合を検出する

## 振舞

```
Given: base / B / C の3ファイルがある
When:  B と C の両方が同じセルを base と異なる値に変更している
And:   B の値 ≠ C の値
Then:  そのセルの conflict が true になる
And:   review_required が true になる
```

```
Given: B と C の両方が同じセルを変更している
When:  B の値 = C の値（同じ変更）
Then:  conflict は false になる
And:   review_required は false になる
And:   採用値は B を優先する（B = C なので結果は同じ）
```
