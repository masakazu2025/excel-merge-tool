# B-002 競合を検出する

## 振舞

```
Given: base / B / C の3ファイルがある
When:  B と C の両方が同じセルを base と異なる値に変更している
And:   B の値 ≠ C の値
Then:  status が "conflict" になる
And:   changed_by が "both" になる
```

```
Given: B と C の両方が同じセルを変更している
When:  B の値 = C の値（同じ変更）
Then:  status は "conflict" にならない（update / add / sub / new のいずれか）
And:   changed_by は "both" になる
And:   採用値は B を優先する（B = C なので結果は同じ）
```
