# B-001 セル値の差分を検出する

## 振舞

```
Given: base.xlsx / file_b.xlsx / file_c.xlsx を用意している
When:  差分抽出を実行する
Then:  全セルについて base_value / b_value / c_value に常に実値を保持する
And:   changed_by フィールドに変更者（b / c / both / null）が設定される
And:   status フィールドに変更種別が設定される
```

## changed_by の判定

| 条件 | changed_by |
|------|-----------|
| b_value ≠ base_value かつ c_value = base_value | `"b"` |
| c_value ≠ base_value かつ b_value = base_value | `"c"` |
| b_value ≠ base_value かつ c_value ≠ base_value | `"both"` |
| b_value = base_value かつ c_value = base_value | `null` |

## 対象セル型

| 型 | XMLタグ | 備考 |
|----|---------|------|
| value | `<v>` | sharedStrings展開後の値 |
| formula | `<f>` | 数式文字列で比較（計算結果は比較しない） |
| date | `<v>` | Excelシリアル値（数値）として比較 |
| rich_text | sharedStrings `<si>` XML全体 | 書式変化も検出対象 |

## 備考

- sharedStrings.xml をパースしてインデックス → 文字列に展開してから比較する
- インライン文字列（`<is>` タグ）も別途処理する
- マージ元（base）に存在するシートを基準とする（B・Cにのみあるシートは対象外）
- 2ファイル比較時（file_c なし）は c_value に null を設定する
