# B-005 図形の差分を検出する

## 振舞（テキストボックスのテキスト変更）

```
Given: base と B/C で同じIDの図形（sp）が存在する
When:  差分抽出を実行する
Then:  テキスト内容の変更が差分として出力される
And:   status が B-003 のルールに従って付与される
```

## 振舞（図形の増減）

```
Given: B または C にのみ存在する図形がある（IDが一致しない）
When:  差分抽出を実行する
Then:  added_b / added_c / deleted_b / deleted_c として出力される
And:   status が "delete" になる
```

## 対象種別

| 種別 | XMLタグ | テキスト比較 | 増減検知 |
|------|---------|------------|---------|
| テキストボックス等 | `<sp>` | ✓ | ✓ |
| 貼り付け画像 | `<pic>` | — | ✓ |
| グラフ | `<graphicFrame>` | — | ✓ |

## 備考

- 図形の対応付けは `<cNvPr id="...">` の id 属性で行う
- `xl/drawings/drawing*.xml`（標準）と `xl/drawings/vmlDrawing*.xml`（旧形式）を両方パースする
- 画像・グラフの中身の変更は検出しない（増減のみ）
