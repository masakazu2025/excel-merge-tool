# B-004 コメントの差分を検出する

## 対象

| 種類 | 保存場所 |
|------|---------|
| ノート（旧形式） | `xl/comments*.xml` |
| スレッドコメント（新形式） | `xl/threadedComments/*.xml` |

## 振舞

```
Given: base / B / C のいずれかにコメント（ノートまたはスレッド）がある
When:  差分抽出を実行する
Then:  コメントの追加・変更・削除が差分として出力される
```

## 判定ルール

セル差分（B-003）と同じ status 判定ロジックを適用する。

| 条件 | status |
|------|--------|
| base になし → B or C にあり | `"new"` |
| base にあり → B or C がなし | `"delete"` |
| テキスト diff が末尾追記のみ | `"add"` |
| テキスト diff が末尾削除のみ | `"sub"` |
| その他の変更 | `"update"` |

## 備考

- スレッドコメントはスレッド内の全返信テキストを結合して1つのテキストとして比較する
- base vs B、base vs C でそれぞれ比較する
