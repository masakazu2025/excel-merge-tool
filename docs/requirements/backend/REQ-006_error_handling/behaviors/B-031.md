# B-031 フロントエンドのエラーハンドリング

## 振舞（F001: ネットワークエラー）

```
Given: サーバーが停止している、またはネットワークが断絶している
When:  fetch() を呼び出す
Then:  fetch が throw する（TypeError: Failed to fetch など）
And:   "サーバーに接続できません" というエラーメッセージを画面に表示する
And:   画面はクラッシュしない
```

## 振舞（F002: バックエンドエラーレスポンス受信）

```
Given: バックエンドが 4xx/5xx ステータスと {"error_code": "Exxx", "message": "..."} を返した
When:  fetch のレスポンスを受け取る
Then:  response.ok が false になる
And:   レスポンスボディの message をそのまま画面に表示する
And:   画面はクラッシュしない
```

## 振舞（F003: response.json() 失敗）

```
Given: レスポンスボディが JSON として不正な文字列である
When:  response.json() を呼び出す
Then:  SyntaxError が throw される
And:   "レポートの形式が正しくありません" というエラーメッセージを画面に表示する
And:   console.error に詳細を出力する
And:   画面はクラッシュしない
```

## 振舞（F003: sheets フィールドが undefined）

```
Given: レスポンスの JSON は valid だが sheets フィールドが存在しない
When:  Report 画面がデータを表示しようとする
Then:  "レポートの形式が正しくありません" というエラーメッセージを画面に表示する
And:   console.error に詳細を出力する
And:   画面はクラッシュしない
```

## 振舞（F003: DiffGrid に不正なセルデータ）

```
Given: cells 配列の要素に cell フィールドが存在しない、または不正な形式である
When:  DiffGrid がセルをレンダリングしようとする
Then:  該当セルをスキップする（画面全体はクラッシュしない）
And:   console.error に詳細を出力する
```
