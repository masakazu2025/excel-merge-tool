# B-030 バックエンドのエラーハンドリング

## 振舞（E001: 対応外ファイル形式）

```
Given: xlsx / xlsm 以外のファイルが compare エンドポイントに送信された
When:  ファイルを受け取る
Then:  HTTP 422 と {"error_code": "E001", "message": "対応していないファイル形式です"} を返す
```

## 振舞（E002: ファイル破損 - ZIP不正）

```
Given: ZIP構造が不正なファイルが渡された（BadZipFile）
When:  zipfile.ZipFile でオープンしようとする
Then:  HTTP 422 と {"error_code": "E002", "message": "ファイルが破損しているか読み込めません"} を返す
```

## 振舞（E002: ファイル破損 - workbook.xml なし）

```
Given: ZIP は開けるが xl/workbook.xml が存在しない
When:  workbook.xml を読み込もうとする
Then:  HTTP 422 と {"error_code": "E002", "message": "ファイルが破損しているか読み込めません"} を返す
```

## 振舞（E002: 0バイトファイル）

```
Given: 0バイトのファイルが渡された
When:  zipfile.ZipFile でオープンしようとする
Then:  HTTP 422 と {"error_code": "E002", "message": "ファイルが破損しているか読み込めません"} を返す
```

## 振舞（E003: 比較可能シートなし - 全シートが B/C に存在しない）

```
Given: base の全シートが B/C のどちらにも存在しない
When:  差分抽出を実行する
Then:  HTTP 422 と {"error_code": "E003", "message": "比較可能なシートが見つかりません"} を返す
```

## 振舞（E003: 比較可能シートなし - 全シートパスが壊れている）

```
Given: workbook.xml にシートが定義されているが、全シートファイルパスが zipfile 内に存在しない
When:  差分抽出を実行する
Then:  HTTP 422 と {"error_code": "E003", "message": "比較可能なシートが見つかりません"} を返す
```

## 振舞（E004: XMLパース失敗）

```
Given: xl/worksheets/sheet1.xml などが壊れた XML を含む
When:  ET.fromstring() が ET.ParseError を送出する
Then:  HTTP 422 と {"error_code": "E004", "message": "ファイルの内容を解析できませんでした"} を返す
```

## 振舞（E004: sharedStrings インデックス超過）

```
Given: セルの sharedStrings インデックスが sharedStrings.xml の件数を超えている
When:  shared_strings[idx] にアクセスする
Then:  HTTP 422 と {"error_code": "E004", "message": "ファイルの内容を解析できませんでした"} を返す
And:   元の IndexError をログに出力する
```

## 振舞（E005: JSONシリアライズ失敗）

```
Given: 差分抽出結果に JSON シリアライズできない値が含まれる
When:  json.dumps() が失敗する
Then:  HTTP 422 と {"error_code": "E005", "message": "比較処理中にエラーが発生しました"} を返す
And:   元の例外メッセージをログに出力する
```

## 振舞（E005: レポート保存失敗）

```
Given: output ディレクトリへの書き込み権限がない、またはディスクが満杯
When:  レポート JSON ファイルを保存しようとする
Then:  HTTP 422 と {"error_code": "E005", "message": "比較処理中にエラーが発生しました"} を返す
And:   元の例外メッセージをログに出力する
```

## 振舞（E005: その他予期せぬエラー）

```
Given: 上記以外のいかなる例外が extract_diff 内で発生した
When:  差分抽出を実行する
Then:  HTTP 422 と {"error_code": "E005", "message": "比較処理中にエラーが発生しました"} を返す
And:   元の例外メッセージをログに出力する
```

## 振舞（E006: レポートが見つからない）

```
Given: 存在しない report_id で GET /api/reports/{report_id} にアクセスした
When:  対応するファイルを探す
Then:  HTTP 404 と {"error_code": "E006", "message": "レポートが見つかりません"} を返す
```

## 振舞（E007: レポートファイル読み込み失敗）

```
Given: report_id に対応するファイルは存在するが読み込みに失敗する
When:  JSON ファイルを open / json.loads する
Then:  HTTP 404 と {"error_code": "E007", "message": "レポートを開けませんでした"} を返す
And:   元の例外メッセージをログに出力する
```

## 振舞（E007: レポートJSON パース失敗）

```
Given: report_id に対応するファイルが不正な JSON を含む
When:  json.loads() が失敗する
Then:  HTTP 404 と {"error_code": "E007", "message": "レポートを開けませんでした"} を返す
And:   元の例外メッセージをログに出力する
```
