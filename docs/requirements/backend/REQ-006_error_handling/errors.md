# エラーコード一覧

## バックエンドエラー（E001〜E007）

| コード | メッセージ | 発生条件 | HTTPステータス |
|--------|----------|---------|--------------|
| `E001` | 対応していないファイル形式です | xlsx / xlsm 以外のファイルが渡された | 422 |
| `E002` | ファイルが破損しているか読み込めません | ZIP構造不正 / ファイルが0バイト / workbook.xml が存在しない | 422 |
| `E003` | 比較可能なシートが見つかりません | base の全シートが B/C に存在しない / 全シートパスが壊れている | 422 |
| `E004` | ファイルの内容を解析できませんでした | XML パース失敗 / sharedStrings インデックス超過など | 422 |
| `E005` | 比較処理中にエラーが発生しました | JSON シリアライズ失敗 / レポート保存失敗 / その他予期せぬエラー | 422 |
| `E006` | レポートが見つかりません | 存在しない report_id へのアクセス | 404 |
| `E007` | レポートを開けませんでした | レポート JSON ファイルの読み込み失敗 / JSON パース失敗 | 404 |

## フロントエンドエラー（F001〜F003）

| コード | メッセージ | 発生条件 |
|--------|----------|---------|
| `F001` | サーバーに接続できません | fetch が throw（ネットワーク断 / サーバー停止） |
| `F002` | （E001〜E007 のメッセージをそのまま表示） | バックエンドから 4xx/5xx が返ったとき |
| `F003` | レポートの形式が正しくありません | response.json() が失敗 / sheets フィールドが undefined / DiffGrid に不正なセルデータ |

## 備考

- E005 はキャッチオール。ログには元の例外メッセージを出力する
- F003 はキャッチオール。開発者向けに console.error で詳細を出力する
- 一部シートが欠けている場合はエラーにせず、対象シートのみ処理して結果に含める
