# B-036 cell-range エンドポイント

## エンドポイント仕様

```
GET /api/reports/{report_id}/cell-range
  ?sheet=Sheet1
  &row=1          # 行番号（rowとcolはどちらか一方）
  &col=A          # 列番号
```

## 振舞1: 指定行のセル値を返す

**Given** 有効なレポートIDと `?sheet=Sheet1&row=1` を指定している
**When** エンドポイントにGETリクエストを送る
**Then** 200 OK で `{ "A": "商品名", "B": "単価", "C": "" }` 形式のJSONを返す
**And** 値は base ファイルのセル値を使用する
**And** 差分がないセルも含めてすべての列を返す

## 振舞2: 指定列のセル値を返す

**Given** 有効なレポートIDと `?sheet=Sheet1&col=A` を指定している
**When** エンドポイントにGETリクエストを送る
**Then** 200 OK で `{ "2": "りんご", "3": "みかん", "4": "" }` 形式のJSONを返す
**And** 値は base ファイルのセル値を使用する
**And** 差分がないセルも含めてすべての行を返す

## 振舞3: rowとcolの両方またはどちらもない場合はエラー

**Given** `?row` と `?col` の両方を指定、またはどちらも指定しない
**When** エンドポイントにGETリクエストを送る
**Then** 400 Bad Request を返す

## 振舞4: 存在しないレポートIDの場合はエラー

**Given** 存在しないレポートIDを指定している
**When** エンドポイントにGETリクエストを送る
**Then** 404 Not Found を返す
