# B-007 3ファイルをアップロードして比較を実行できる

## 振舞

```
Given: base.xlsx / file_b.xlsx / file_c.xlsx を用意している
When:  POST /api/compare に multipart/form-data で3ファイルを送信する
Then:  差分抽出が実行される
And:   output/{timestamp}_diff.json が保存される
And:   レポートIDとメタ情報がレスポンスとして返る
```

## リクエスト

```
POST /api/compare
Content-Type: multipart/form-data

base_file: base.xlsx
file_b:    file_b.xlsx
file_c:    file_c.xlsx
```

## レスポンス

```json
{
  "report_id": "20260314_153042",
  "created_at": "2026-03-14T15:30:42",
  "base_file": "base.xlsx",
  "file_b": "file_b.xlsx",
  "file_c": "file_c.xlsx",
  "total_diffs": 42,
  "total_conflicts": 3
}
```
