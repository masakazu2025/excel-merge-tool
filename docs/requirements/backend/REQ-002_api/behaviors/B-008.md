# B-008 過去レポート一覧を取得できる

## 振舞

```
Given: output/ に1件以上の diff.json が保存されている
When:  GET /api/reports?page=1&limit=10 を呼ぶ
Then:  items にレポートのメタ情報一覧が新しい順で返る
And:   total に全件数、page に現在ページが含まれる
```

```
Given: output/ に diff.json が1件もない
When:  GET /api/reports を呼ぶ
Then:  items が空配列、total が 0 で返る
```

```
Given: output/ に5件の diff.json がある
When:  GET /api/reports?page=2&limit=2 を呼ぶ
Then:  items に2件（2ページ目）が返る
And:   total が 5、page が 2 で返る
```

## クエリパラメータ

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `page`    | 1         | ページ番号（1始まり） |
| `limit`   | 10        | 1ページあたりの件数 |

## レスポンス

```json
{
  "items": [
    {
      "report_id": "20260314_153042",
      "created_at": "2026-03-14T15:30:42",
      "base_file": "base.xlsx",
      "file_b": "file_b.xlsx",
      "file_c": "file_c.xlsx",
      "total_diffs": 42,
      "total_conflicts": 3
    }
  ],
  "total": 1,
  "page": 1
}
```

## 備考

- `output/` 内の `*_diff.json` をファイル名降順（新しい順）で返す
- メタ情報は各JSONファイル内の `meta` キーから取得する
