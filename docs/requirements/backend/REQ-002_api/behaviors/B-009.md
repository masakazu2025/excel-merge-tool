# B-009 指定レポートの内容を取得できる

## 振舞

```
Given: output/{report_id}_diff.json が存在する
When:  GET /api/reports/{report_id} を呼ぶ
Then:  diff.json の内容がそのまま返る
```

```
Given: 指定した report_id の diff.json が存在しない
When:  GET /api/reports/{report_id} を呼ぶ
Then:  404 エラーが返る
```
