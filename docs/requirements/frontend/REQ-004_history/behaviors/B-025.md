# B-025 ページネーションで20件ずつ表示する

## 振舞

```
Given: 履歴が20件を超えている
When:  履歴画面を表示する
Then:  1ページあたり20件を表示し、前へ/次へボタンでページを切り替えられる

Given: 先頭ページを表示している
When:  ページネーションを確認する
Then:  「前へ」ボタンがグレーアウト（非活性）

Given: 末尾ページを表示している
When:  ページネーションを確認する
Then:  「次へ」ボタンがグレーアウト（非活性）
```

## 備考

- API: `GET /api/reports?page=1&limit=20`
- レスポンス: `{ items: [...], total: number, page: number, limit: number }`
