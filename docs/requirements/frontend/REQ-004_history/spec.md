# REQ-004 比較履歴一覧画面

## 概要

過去の比較結果をテーブル形式で一覧表示する画面。行クリックで結果画面に遷移する。

## レイアウト

```
┌──────────────────────────────────────────────────────┐
│ 比較履歴                                              │
├────────────────┬──────────┬────────────┬─────────────┤
│ 実行日時        │ 比較元   │ 変更ファイル │ 差分件数    │
├────────────────┼──────────┼────────────┼─────────────┤
│ 2026-03-14     │ base.xlsx│ file1.xlsx  │ 42件        │
│ 15:30          │          │ file2.xlsx  │             │
├────────────────┼──────────┼────────────┼─────────────┤
│ 2026-03-10     │ base.xlsx│ file1.xlsx  │ 15件        │
│ 09:12          │          │             │             │
└────────────────┴──────────┴────────────┴─────────────┘
```

## 受け入れ条件

- [x] 履歴をテーブル形式で表示できる
- [x] 実行日時・比較元・変更ファイル・差分件数を表示する
- [x] 変更ファイルは1件・2件どちらにも対応する
- [x] 履歴が0件のときは空メッセージを表示する
- [x] 行クリックで結果画面（/report/:id）に遷移する
- [x] ページネーションで20件ずつ表示する（前へ/次へ）

## ページネーション仕様

- 1ページあたり20件固定
- API: `GET /api/reports?page=1&limit=20` → `{ items, total, page, limit }`
- 先頭ページで「前へ」、末尾ページで「次へ」はグレーアウト
