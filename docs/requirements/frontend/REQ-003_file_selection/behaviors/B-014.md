# B-014 ファイル名の省略表示とマウスオーバー表示

## 振舞

```
Given: ファイル名がリストの表示幅を超える長さである
When:  リストを表示する
Then:  ファイル名が最大幅で省略され末尾が「...」になる

When:  そのファイル名にマウスオーバーする
Then:  ツールチップでフルネームが表示される
```

## 備考

- CSS の `text-overflow: ellipsis` で実装する
- ツールチップは `title` 属性で実装する
