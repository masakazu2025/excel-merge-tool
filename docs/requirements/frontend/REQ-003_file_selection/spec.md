# REQ-003 ファイル選択画面

## 概要

比較対象のExcelファイルを選択し、比較元を指定して比較を実行する画面。
ボタンによるダイアログ選択とドラッグ&ドロップの両方に対応する。
対応ファイル形式: `.xlsx` / `.xlsm`

## レイアウト

```
┌────────────────────────────────────────────┐
│ [+ ファイルを追加] or ドロップ              │
├──────────────────────────────────┬─────────┤
│ ◉  📄 base.xlsx  [比較元]        │  [×]   │
│ ○  📄 file1.xlsx                 │  [×]   │
│ ○  📄 file2.xlsx                 │  [×]   │
├──────────────────────────────────┴─────────┤
│                          [比較実行]         │
└────────────────────────────────────────────┘
```

## 受け入れ条件

- [x] ボタンからダイアログでファイルを選択できる（.xlsx / .xlsm）
- [x] ドロップゾーンにDnDでファイルを追加できる（.xlsx / .xlsm）
- [x] 複数ファイルを一度に追加できる
- [x] 追加したファイルがリストに表示される
- [x] ファイル名が長い場合は省略表示、マウスオーバーでフルネームを表示する
- [x] デフォルトで先頭行が比較元に選択される
- [x] ラジオボタンまたはファイル行クリックで比較元を変更できる
- [x] 選択中の比較元行に [比較元] バッジが表示される
- [x] ×ボタンでファイルをリストから削除できる
- [x] 比較元行を削除すると先頭行が自動的に比較元になる
- [x] 1ファイル以下のとき比較実行ボタンがグレーアウトする

## デザイン仕様

- ラジオボタン: 未選択時は白背景・グレー枠、選択時は青
