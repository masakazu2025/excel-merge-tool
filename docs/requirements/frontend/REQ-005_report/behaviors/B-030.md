# B-030 差分なしシートのタブを非表示にする

## 振舞

```
Given: 差分データに差分0件のシートが含まれる
When:  レポート画面を表示する
Then:  差分0件のシートのタブは表示されない
       差分ありのシートのタブのみ表示される
```

```
Given: 全シートが差分0件である
When:  レポート画面を表示する
Then:  「差分がありません」メッセージが表示される
```

## 備考

- バックエンドのAPIレスポンスは変更しない（フロントエンドでフィルタリング）
- 初期表示シートは差分ありの最初のシートとする
