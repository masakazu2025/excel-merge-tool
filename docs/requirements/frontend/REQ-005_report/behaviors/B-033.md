# B-033 グリッドエリアの全画面化とスクロールバー常時表示

## 振舞1: グリッドエリアが画面全体を使う

**Given** 差分比較結果を表示している
**When** レポート画面を開く
**Then** 差分グリッドがビューポートの高さ・幅を最大限使って表示される（Excelの最大化と同じイメージ）
**And** ヘッダー・シートタブ・フィルタバーは固定表示を維持する
**And** グリッドエリアの高さ = ビューポート高さ - ヘッダー - シートタブ - フィルタバー
**And** グリッドエリアの幅 = ビューポート幅いっぱい

## 振舞2: 縦・横スクロールバーが常時表示される

**Given** 差分グリッドが表示されている
**When** グリッドのコンテンツがエリア内に収まっている場合でも
**Then** 縦・横スクロールバーが常時表示される
**And** ユーザーがスクロール可能な範囲を一目で把握できる
