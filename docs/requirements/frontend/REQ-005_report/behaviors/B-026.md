# B-026 グリッドで差分セルを色分け表示する

## 振舞

```
Given: 差分データが存在する
When:  結果画面を表示する
Then:  変更種別に応じた色でセルが表示される
```

## 色定義

| 色 | 状態 | 条件 |
|----|------|------|
| オレンジ | コンフリクト | conflict=true |
| 緑 | 新規追加 | diff_hint=new |
| 青 | 片側のみ追加 | diff_hint=insert_only |
| 赤 | 削除 | diff_hint=delete_only |
| 黄 | 変更 | diff_hint=replace |
| 紫 | 日付更新 | diff_hint=newer_date |

## 備考

- セルの表示値は b_value（null の場合は base_value）を使用
- 画面右上に凡例を表示する
