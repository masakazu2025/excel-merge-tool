# B-028 モーダルを開いたままキーボードで差分セル間を移動できる

## 振舞

```
Given: セルをクリックして詳細モーダルが開いている
When:  矢印キーを押す
Then:  モーダルを開いたまま隣接する差分セルへ移動し、モーダル内容が切り替わる
```

## キー操作

| キー | 動作 |
|------|------|
| → | 同じ行の右にある次の変更セルへ移動。端で停止 |
| ← | 同じ行の左にある前の変更セルへ移動。端で停止 |
| ↓ | 次の行へ移動。現在の列に最も近い変更セルへスナップ |
| ↑ | 前の行へ移動。現在の列に最も近い変更セルへスナップ |
| Esc | モーダルを閉じる |

## 備考

- 矢印キーはモーダルが開いているときのみ有効（グリッド単体ではキー操作なし）
- ↓↑ で対象行に変更セルがない場合は、その行をスキップして次の行へ
