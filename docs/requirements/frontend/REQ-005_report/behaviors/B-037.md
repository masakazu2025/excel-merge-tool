# B-037 列名・行名の表示設定

## 振舞1: 表示設定ポップアップを開く

**Given** 差分グリッドが表示されている
**When** フィルタバーの「⚙ 表示設定」ボタンをクリックする
**Then** ポップアップが表示される
**And** 「列名に使う行番号」入力欄と「行名に使う列番号」入力欄が表示される

## 振舞2: 列名を設定する

**Given** 表示設定ポップアップが開いている
**When** 「列名に使う行番号」に `1` を入力して適用する
**Then** APIから行1のセル値を取得する
**And** グリッドの列ヘッダー下に名称行が挿入される
**And** 各列の名称行には対応するセル値が表示される（値がない場合は空白）

## 振舞3: 行名を設定する

**Given** 表示設定ポップアップが開いている
**When** 「行名に使う列番号」に `A` を入力して適用する
**Then** APIから列Aのセル値を取得する
**And** グリッドの行番号の隣に名称列が挿入される
**And** 各行の名称列には対応するセル値が表示される（値がない場合は空白）

## 振舞4: 設定をクリアする

**Given** 列名または行名が設定されている
**When** ポップアップの「クリア」ボタンをクリックする
**Then** 列名・行名の表示が解除される

## 振舞5: シート切り替えで設定がリセットされる

**Given** 列名・行名が設定されている
**When** シートを切り替える
**Then** 列名・行名の設定がリセットされる

## グリッド表示イメージ

```
|      |  A      |  B    |   ← 列ヘッダー（常時表示）
|      | 商品名  |  単価  |   ← 列名行（設定時のみ）
|  2  | りんご  |  ...   |   ← 行名列（設定時のみ）
|  3  | みかん  |  ...   |
```
