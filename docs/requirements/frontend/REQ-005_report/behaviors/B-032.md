# B-032 詳細モーダルのサイズ固定とセル表示の統一

## 振舞1: モーダルサイズが固定されている

```
Given: 差分詳細モーダルが開いている
When:  キーボードで別の差分セルに移動する
Then:  モーダルのサイズは変わらない
       コンテンツがモーダルの高さを超える場合はスクロールバーで対応する
```

## 振舞2: 2ファイル比較では常に2セル表示

```
Given: 2ファイル比較（base + B）の差分詳細モーダルが開いている
When:  どの差分セルを表示しても
Then:  Base と B の2セルが常に並んで表示される
       値が null のセルは「（空）」と表示される
```

## 振舞3: 3ファイル比較では常に3セル表示

```
Given: 3ファイル比較（base + B + C）の差分詳細モーダルが開いている
When:  どの差分セルを表示しても
Then:  Base・B・C の3セルが常に並んで表示される
       値が null のセルは「（空）」と表示される
```

## 備考

- モーダルの固定サイズは高さ・幅ともに一定とする
- null と空文字はユーザー視点では同じ意味のため、どちらも「（空）」と表示する」
