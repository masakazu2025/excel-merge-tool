# B-038 比較テーブルでキーボードナビゲーション（モーダルなし）

## 概要

比較結果グリッドにフォーカスしている間、モーダルを開かずに矢印キーでセル間を移動できる。
Enter キーで選択中のセルのモーダルを開く。

## Given / When / Then

### Given: 比較結果グリッドが表示された直後

**When:** グリッドがマウントされる（シート切替含む）
**Then:** 先頭の差分セル（左上）にカーソルが自動配置される
**And:** グリッドコンテナが自動フォーカスされ、すぐにキーボード操作が有効になる

---

### Given: 比較結果グリッドがフォーカス状態

**When:** `→` キーを押す
**Then:** 現在の行内で右の差分セルにカーソルが移動する（モーダルは開かない）

**When:** `←` キーを押す
**Then:** 現在の行内で左の差分セルにカーソルが移動する

**When:** `↓` キーを押す
**Then:** 次の行の差分セルにカーソルが移動する（現在の列に最も近い列）

**When:** `↑` キーを押す
**Then:** 前の行の差分セルにカーソルが移動する

**When:** `Enter` キーを押す
**Then:** フォーカス中のセルのモーダルが開く

---

### Given: モーダルが開いている状態

**When:** 矢印キーを押す
**Then:** 次のセルにカーソルが移動し、モーダルの内容も更新される（B-028 の挙動と同じ）

---

## やらないこと

- グリッド外でのキーボード操作
- グリッド未フォーカス時のキーボード操作
