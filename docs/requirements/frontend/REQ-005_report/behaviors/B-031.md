# B-031 列・行フィルタで特定の列・行を除外できる

## 振舞1: 列を除外する

```
Given: 差分グリッドが表示されている
When:  「列 ▾」ドロップダウンを開き、列のチェックを外す
Then:  該当列に属するセルが非表示になる
       該当列しか差分を持たない行もあわせて非表示になる
       ボタンが「列 N除外 ▾」とオレンジ色になる
```

## 振舞2: 行を除外する

```
Given: 差分グリッドが表示されている
When:  「行 ▾」ドロップダウンを開き、行のチェックを外す
Then:  該当行に属するセルが非表示になる
       該当行しか差分を持たない列もあわせて非表示になる
       ボタンが「行 N除外 ▾」とオレンジ色になる
```

## 振舞3: すべて解除

```
Given: 列または行が除外されている
When:  ドロップダウン内の「すべて解除」をクリックする
Then:  その種別（列 or 行）の除外がすべてクリアされる
       グリッドが元の表示に戻る
       ボタンが通常のグレーに戻る
```

## 振舞4: フィルタ状態はシートごとに維持される

```
Given: Sheet1 で列を除外している
When:  Sheet2 に切り替える
Then:  Sheet2 のフィルタは初期状態（除外なし）
       Sheet1 に戻ると除外状態が維持されている
```

## 振舞5: レポートを離れるとフィルタがクリアされる

```
Given: 列または行が除外されている
When:  ホームへ戻る・履歴画面へ移動・新規比較を実行する
Then:  すべてのシートのフィルタ状態がクリアされる
```

## 備考

- フィルタ対象は差分のある列・行のみ（ドロップダウンに表示する項目）
- バックエンドの比較結果は変更しない（フロントエンドのみで絞り込む）
- フィルタ状態はシートごとに個別管理し、同一レポートを開いている間は維持する
- レポートから離脱（ホーム・履歴・新規比較）でクリアされる
