# B-035 Ctrl+ホイールでグリッドをズームできる

## 振舞1: Ctrl+ホイールで縮尺が変わる

**Given** 差分グリッドが表示されている
**When** Ctrl キーを押しながらマウスホイールを操作する
**Then** グリッドの縮尺が変わる
**And** ホイールアップで拡大（最大100%）、ホイールダウンで縮小（最小50%）
**And** ステップは10%刻み

## 振舞2: 縮尺インジケーターが表示される

**Given** 縮尺が100%未満の状態
**When** グリッドを表示している
**Then** 現在の縮尺（例: 80%）がフィルタバーに表示される

## 振舞3: 縮尺リセット

**Given** 縮尺が変更されている
**When** 縮尺インジケーターをクリックする
**Then** 縮尺が100%に戻る

## 備考

- CSS `zoom` プロパティをグリッドコンテナに適用する
- Ctrl+ホイール時はページスクロールを抑制する（`e.preventDefault()`）
