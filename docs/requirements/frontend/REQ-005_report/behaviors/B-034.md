# B-034 フィルタ操作強化

## 振舞1: 列ヘッダーのダブルクリックで除外

**Given** 差分グリッドが表示されている
**When** 列ヘッダー（A, B, C...）をダブルクリックする
**Then** その列がフィルタ除外される
**And** B-031の通常の除外と同じ動作をする（セル非表示・相互絞り込み等）

## 振舞2: 行ヘッダーのダブルクリックで除外

**Given** 差分グリッドが表示されている
**When** 行ヘッダー（行番号）をダブルクリックする
**Then** その行がフィルタ除外される

## 振舞3: フィルタ一括解除ボタン

**Given** 列または行のフィルタが1件以上適用されている
**When** フィルタバーの「すべて解除」ボタンをクリックする
**Then** 列・行フィルタがすべてリセットされる
**And** ボタンはフィルタが1件もない場合は非表示にする

## 振舞4: ドロップダウン内スクロール

**Given** 差分のある列または行が多数ある
**When** フィルタドロップダウンを開く
**Then** リストに最大高さが設定されており、超過時はスクロールできる

## 振舞5: ドロップダウン内検索

**Given** フィルタドロップダウンが開いている
**When** 検索ボックスに文字を入力する
**Then** 列名または行番号が入力文字列に前方一致する項目のみ表示される
**And** 検索ボックスをクリアすると全項目が再表示される
