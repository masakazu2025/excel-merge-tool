# REQ-005 結果画面

## 概要

差分比較結果をExcelライクなグリッドで表示する画面。
変更があった行・列のみ表示し、色でセルの状態を視覚的に区別する。

## レイアウト

```
┌─────────────────────────────────────────────────────┐
│ ← ホーム │ base: base.xlsx  B: file_b.xlsx  差分17件 │  ← ヘッダー（固定）
├─────────────────────────────────────────────────────┤
│ [Sheet1] [Sheet3]                                   │  ← シートタブ（差分ありのみ表示）
├─────────────────────────────────────────────────────┤
│ フィルタ: [すべて] [コンフリクト] [要確認]  3件      │
├──────┬──────┬──────┬──────┬──────┬──────────────────┤
│      │  A   │  B   │  D   │  E   │  F               │  ← 列（変更ある列のみ）
├──────┼──────┼──────┼──────┼──────┼──────────────────┤
│   3  │ [緑] │      │      │      │                  │  ← 行（変更ある行のみ）
│   4  │      │ [黄] │      │      │                  │
│   7  │      │      │ [橙] │      │                  │
│  12  │      │      │      │ [黄] │                  │
└──────┴──────┴──────┴──────┴──────┴──────────────────┘
```

## 色定義

| 色 | 状態 | diff_hint / 条件 |
|----|------|-----------------|
| オレンジ | コンフリクト | conflict=true |
| 緑 | 新規追加 | new |
| 青 | B/Cのみ追加 | insert_only |
| 赤 | 削除 | delete_only |
| 黄 | 変更 | replace |
| 紫 | 日付更新 | newer_date |

## セル表示値

- b_value が null でない場合 → b_value を表示
- b_value が null の場合 → base_value を表示

## キーボード操作

- 矢印キー: 差分セル間を移動（行・列の順に次の変更セルへ）

## 列・行ヘッダー

- 列: アルファベット（A, B, C...）※後から名称定義を追加予定
- 行: 行番号（Excel準拠）

## 受け入れ条件

- [x] 変更のある行・列のみ表示する
- [x] セルを色で状態区別できる（凡例付き）
- [x] セルクリックで詳細モーダルを表示する（base / B / C 値を並列表示）
- [x] モーダルを開いたまま矢印キーで変更セル間を移動できる
- [x] シートを切り替えられる
- [x] 差分なしシートのタブは表示しない（全シート差分なしの場合はメッセージ表示）
- [x] フィルタで表示絞り込みができる（すべて/コンフリクト/要確認）
- [ ] 列ヘッダーの名称定義（後から追加予定）
