# REQ-005 結果画面

## 概要

差分比較結果をExcelライクなグリッドで表示する画面。
変更があった行・列のみ表示し、色でセルの状態を視覚的に区別する。

## レイアウト

```
┌─────────────────────────────────────────────────────┐
│ ← ホーム │ base: base.xlsx  B: file_b.xlsx  差分17件 │  ← ヘッダー（固定）
├─────────────────────────────────────────────────────┤
│ [Sheet1] [Sheet3]                                   │  ← シートタブ（差分ありのみ表示）
├─────────────────────────────────────────────────────┤
│ フィルタ: [すべて] [コンフリクト] [要確認]  3件  [列 ▾] [行 ▾] │
├──────┬──────┬──────┬──────┬──────┬──────────────────┤
│      │  A   │  B   │  D   │  E   │  F               │  ← 列（変更ある列のみ）
├──────┼──────┼──────┼──────┼──────┼──────────────────┤
│   3  │ [緑] │      │      │      │                  │  ← 行（変更ある行のみ）
│   4  │      │ [黄] │      │      │                  │
│   7  │      │      │ [橙] │      │                  │
│  12  │      │      │      │ [黄] │                  │
└──────┴──────┴──────┴──────┴──────┴──────────────────┘
```

## 色定義

| 色 | 状態 | diff_hint / 条件 |
|----|------|-----------------|
| オレンジ | コンフリクト | conflict=true |
| 緑 | 新規追加 | new |
| 青 | B/Cのみ追加 | insert_only |
| 赤 | 削除 | delete_only |
| 黄 | 変更 | replace |
| 紫 | 日付更新 | newer_date |

## セル表示値

- b_value が null でない場合 → b_value を表示
- b_value が null の場合 → base_value を表示

## キーボード操作

- 矢印キー: 差分セル間を移動（行・列の順に次の変更セルへ）
- Enter: フォーカス中のセルのモーダルを開く
- グリッド表示時に先頭セルへ自動フォーカス

## 列・行ヘッダー

- 列: アルファベット（A, B, C...）※後から名称定義を追加予定
- 行: 行番号（Excel準拠）

## 受け入れ条件

- [x] 変更のある行・列のみ表示する
- [x] セルを色で状態区別できる（凡例付き）
- [x] セルクリックで詳細モーダルを表示する（base / B / C 値を並列表示）
- [x] 詳細モーダルのサイズを固定し、内容超過時はスクロール
- [x] 詳細モーダルは常に比較ファイル数分のセルを表示する（null は「（なし）」と明示）
- [x] モーダルを開いたまま矢印キーで変更セル間を移動できる
- [x] シートを切り替えられる
- [x] 差分なしシートのタブは表示しない（全シート差分なしの場合はメッセージ表示）
- [x] フィルタで表示絞り込みができる（すべて/コンフリクト/要確認）
- [x] 列・行フィルタで特定の列・行を除外できる
- [x] グリッドの列・行ヘッダーをダブルクリックでフィルタ除外できる
- [x] フィルタを一括解除できる
- [x] フィルタドロップダウン内をスクロールできる
- [x] フィルタドロップダウン内で検索できる
- [x] 列名・行名をExcelのヘッダー行/列から設定して表示できる
- [x] Ctrl+ホイールでグリッドの縮尺を変更できる（50%〜100%）
- [x] グリッドエリアを画面全体に広げて表示できる
- [x] 縦・横スクロールバーを常時表示する
- [x] グリッドフォーカス中にキーボードでセル移動できる（モーダルなし）
- [x] グリッド表示時に先頭セルへ自動フォーカスされる

## 列・行フィルタ仕様

- フィルタバーに「列 ▾」「行 ▾」ドロップダウンを追加
- ドロップダウン内に差分のある列（または行）のチェックボックス一覧を表示
- 「すべて選択」「すべて解除」ボタンを提供
- チェックを外した列・行に属するセルを非表示にする
- 除外により差分セルが0になった行・列もあわせて非表示にする
- 除外中はボタンをオレンジ色にし「列 N除外 ▾」と件数を表示する
- フィルタ状態はシートごとに個別管理し、同一レポートを開いている間は維持する
- レポートから離脱（ホームへ戻る・履歴画面・新規比較）でクリアされる
- グリッドの列ヘッダー・行ヘッダーをダブルクリックするとその列・行を除外する
- フィルタバーに「すべて解除」ボタンを表示し、列・行フィルタをまとめてリセットできる
- ドロップダウン内にスクロールを設ける（最大高さを設け超過時はスクロール）
- ドロップダウン内に検索ボックスを設け、列名・行番号で絞り込める
