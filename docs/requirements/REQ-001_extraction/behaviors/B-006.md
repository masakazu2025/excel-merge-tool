# B-006 カスタムルール（merge_rules.yaml）を適用する

## 振舞（除外文字列）

```
Given: merge_rules.yaml に excluded_values が設定されている
When:  セル値が excluded_values と完全一致する
Then:  そのセル値を null とみなして比較する
And:   テンプレート文字列への新規入力が diff_hint: "new" として扱われる
```

## 振舞（ルール適用優先度）

```
Given: シート・列・セルの各粒度でルールが定義されている
When:  差分判定を行う
Then:  cell > column > sheet > global の優先度でルールが適用される
```

## 対応ルール種別

| rule | 内容 |
|------|------|
| `date_newer` | 新しい日付への変更をマージ可とする |
| `status_transition` | 定義した遷移方向のみマージ可とする |

## 備考

- 設定ファイルパス: `config/merge_rules.yaml`
- 除外文字列はグローバル値とルール単位の値の和集合が適用される
