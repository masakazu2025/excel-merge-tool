# B-003 diff_hint と review_required を付与する

## 振舞

```
Given: 差分が検出されたセルがある
When:  差分抽出を実行する
Then:  各差分に diff_hint と review_required が付与される
```

## 判定ルール

| 条件 | review_required | diff_hint |
|------|----------------|-----------|
| conflict: true（B≠C） | true | 各種別の値 |
| base が空 → B or C に値あり | false | `new` |
| テキスト diff が追加のみ | false | `insert_only` |
| テキスト diff が削除のみ | true | `delete_only` |
| テキスト diff に削除＋追加混在 | true | `replace` |
| 日付が新しくなる変更（競合なし） | false | `newer_date` |
| 日付が古くなる変更 | true | `replace` |
| 数式の変更 | true | `replace` |

## 備考

- `newer_date` の自動マージ可判定はオプション設定（デフォルト: 有効）
- テキスト diff 判定は Python の `difflib.SequenceMatcher` を使用
