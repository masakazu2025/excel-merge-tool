# B-001 セル値の差分を検出する

## 振舞

```
Given: base.xlsx / file_b.xlsx / file_c.xlsx を用意している
When:  差分抽出を実行する
Then:  base と異なる値を持つセルが差分として出力される
And:   b_value に B の変更値、c_value に C の変更値が入る
And:   変更がない側は null になる
And:   B と C の両方が base と異なり B ≠ C の場合、conflict: true になる
```

## 対象セル型

| 型 | XMLタグ | 備考 |
|----|---------|------|
| value | `<v>` | sharedStrings展開後の値 |
| formula | `<f>` | 数式文字列で比較（計算結果は比較しない） |
| date | `<v>` | Excelシリアル値（数値）として比較 |
| rich_text | sharedStrings `<si>` XML全体 | 書式変化も検出対象 |

## 備考

- sharedStrings.xml をパースしてインデックス → 文字列に展開してから比較する
- インライン文字列（`<is>` タグ）も別途処理する
- マージ元（base）に存在するシートを基準とする（B・Cにのみあるシートは対象外）
