# はじめに

## 動作環境

- Windows 10 / 11
- .xlsx / .xlsm 形式の Excel ファイル

---

## 一般ユーザー向け：配布パッケージを使う

### インストール

1. 配布された zip ファイルを任意のフォルダに展開する
2. `excel-merge-tool/` フォルダが生成される（インストール不要）

### 起動方法

1. `excel-merge-tool/excel-merge-tool.exe` をダブルクリックする
2. コンソールウィンドウが開き、ブラウザが自動的に起動する（`http://localhost:8080`）
3. ブラウザが起動しない場合は手動で `http://localhost:8080` を開く

### 操作方法

[usage.md](usage.md) を参照。

### 終了方法

コンソールウィンドウを閉じる。
