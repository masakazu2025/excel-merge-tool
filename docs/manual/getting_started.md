# はじめに

## 動作環境

- Windows 10 / 11
- .xlsx / .xlsm 形式の Excel ファイル

## インストール

1. `excel-merge-tool.exe` をダウンロードする
2. 任意のフォルダに配置する（インストール不要）

## 起動方法

1. `excel-merge-tool.exe` をダブルクリックする
2. ブラウザが自動的に開く（`http://localhost:8080`）
3. 起動しない場合はブラウザで `http://localhost:8080` を手動で開く

## 終了方法

タスクバーのアイコンを右クリックして「終了」を選択するか、
コマンドプロンプトウィンドウを閉じる。
