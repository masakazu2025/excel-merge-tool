# 操作方法

## 比較の実行

1. **ホーム画面**でファイルを追加する
   - ドロップゾーンに .xlsx / .xlsm ファイルをドラッグ＆ドロップ
   - または「ファイルを選択」クリックでダイアログから選択
   - 2〜3ファイルを追加する

2. **比較元**を選択する
   - ラジオボタンまたは行クリックで比較元（Base）を指定
   - 先頭ファイルがデフォルトで選択される

3. **「比較実行」**ボタンをクリックする

## 結果の確認

- **グリッド表示**：変更セルが色分けで一覧表示される
- **セルをクリック**：詳細モーダルで Base / B / C の値を比較できる
- **矢印キー**：モーダルを開いたまま隣接する変更セルへ移動できる
- **フィルター**：「すべて」「競合」「Bの変更」「Cの変更」で絞り込み

## 色の意味

| 色 | 意味 |
|----|------|
| オレンジ | 競合（B と C が異なる値に変更） |
| 緑 | 新規追加 |
| 青 | 末尾に追記 |
| 赤 | 削除 |
| 薄赤 | 末尾から削除 |
| 黄 | 更新 |

## 履歴

過去の比較結果は「履歴」から再確認できる。
