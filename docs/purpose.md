# 目的

3つのExcelファイル（ベース・B・C）の差分を人間が確認・選択してマージできるWebアプリを提供する。

## 背景

同一のExcelファイルを複数人が並行編集した場合、変更を手作業でマージするのは困難でミスが起きやすい。
本ツールはその作業を補助し、人間の判断が必要な箇所を明確にする。

## 対象ファイル

- 形式: `.xlsx`
- 想定サイズ: 約5MB
- 想定シート数: 20シート程度

## 比較対象

- セル（値・数式・日付・リッチテキスト）
- 標準コメント
- 図形・オブジェクト（テキストボックス・画像・グラフ）
